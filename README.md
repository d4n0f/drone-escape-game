# Drone Escape

Drone Escape is a 3D game made in Unity where the player controls a drone and must survive while collecting coins and avoiding enemy birds.

The player has limited health and time to complete the objective. Batteries can be collected to restore health, while enemy birds constantly chase the player throughout the game.

## Features
* Drone movement system
* Enemy AI with player tracking
* Coin and battery pickups
* Health and timer systems
* Win and lose conditions
* Main menu and pause functionality
* Easy and Hard difficulty modes
* Sound effects and animations
* Gameplay UI
## Controls
* WASD / Arrow Keys — Move
* P — Pause game
* ESC — Restart game
## Built With
* Unity
* C#
* Unity Asset Store assets
